## 🚀 Sobre o Projeto

Este repositório documenta minha jornada de aprendizado em **Python do zero**, reunindo exercícios, práticas e conceitos fundamentais da programação.

O objetivo é construir uma base sólida em lógica de programação, preparando o caminho para áreas mais avançadas como:

- 📊 Ciência de Dados  
- 🤖 Inteligência Artificial  
- 🧱 Engenharia de Dados  

---

## 🧠 Objetivo

✔️ Aprender lógica de programação  
✔️ Desenvolver pensamento computacional  
✔️ Praticar resolução de problemas  
✔️ Criar base sólida para projetos futuros  

---

## ⚙️ Conteúdos Abordados

- 🔢 Variáveis e tipos de dados  
- 🔁 Estruturas de repetição (for / while)  
- 🔀 Estruturas condicionais (if / else)  
- 📦 Funções  
- 🧮 Operadores lógicos e matemáticos  
- 🧩 Resolução de exercícios
